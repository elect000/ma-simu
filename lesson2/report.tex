\subsection{}
\label{sec:org7fbc936}
x 軸の範囲,分割点数および係数をパラメータとして多項式関数の描画を行う Scilab の関数を作成せよ.\\
\begin{minted}[frame=lines,linenos=true,obeytabs,tabsize=4]{Scilab}
function [] = createGraph(xfrom, xto, m, a)
x = linspace(xfrom, xto, m);
y = zeros(1, m);
n = size(a, 2) - 1;
for i=1:n
    y = (y + a(i)) .* x;
end
y = y + a(n + 1);
plot(x, y)
endfunction
\end{minted}
